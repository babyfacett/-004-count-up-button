# 04 カウントアップボタン

このアプリは、ボタンをクリックするたびに数値が1ずつ増えていくシンプルなカウンターです。Streamlit の `session_state` 機能を使って状態を保持するため、ページを更新しなくてもクリック数が保持されます。

## 使い方

1. 以下のコマンドでアプリを起動します。

    ```bash
    streamlit run app.py
    ```

2. アプリに表示される「カウントアップ」ボタンをクリックすると、カウントが 1 ずつ増加します。現時点のカウント値は画面上に表示されます。

3. 必要に応じて、カウンターをリセットする「リセット」ボタンも用意しています。

このアプリは、状態を保持する方法やボタンの使い方を学ぶのに適したサンプルです。
